# 📊 Hotel Booking Analytics Dashboard

## 📌 Overview
This project presents an end-to-end data analytics workflow for analyzing hotel booking performance between 2018 and 2020. The goal is to uncover actionable insights related to revenue generation, cancellation behavior, and customer segmentation within the hospitality industry.

The dashboard is built using **Power BI**, with data preprocessing and transformation performed in **Microsoft Excel and SQL Server**.

---

## 🎯 Objectives
- Analyze hotel booking trends over time  
- Evaluate revenue performance and potential revenue loss  
- Understand customer behavior and segmentation  
- Support data-driven decision making  

## ⚙️ Tools
- **Microsoft Excel** → Data exploration 
- **Microsoft SQL Server** → Data cleaning & transformation  
- **Power BI** → Dashboard & visualization  
 
---

## 📊 Key Performance Indicators (KPIs)
- Total Revenue  
- Potential Revenue (Lost Revenue due to cancellations)  
- Total Bookings  
- Canceled Bookings  
- Average Daily Rate (ADR)  
- Average Length of Stay (ALOS)  
- Average Lead Time  

---

## 📈 Dashboard Features
-  Revenue trends by month  
-  Comparison between City Hotel vs Resort Hotel  
-  Guest distribution map  
-  Market segment analysis  
-  Cancellation impact analysis  

---

## 🔍 Key Insights
- Online Travel Agencies contribute significantly to bookings  
- High cancellation rates impact potential revenue  
- Different patterns exist between City and Resort hotels  
- Guest distribution is globally spread  
- ADR and Length of Stay vary across segments  

---

## 🚀 How to Use
1. Download the `.pbix` file from repository
